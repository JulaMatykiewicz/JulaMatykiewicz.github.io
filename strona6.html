Program w C++ składa się z następujących elementów:
Klasa główna to punkt wejścia do programu. Zawiera ona funkcję main(), która jest wykonywana jako pierwsza.

        C++
        #include <iostream>

        int main() {
          // ...
          return 0;
        }
Funkcje to bloki kodu, które mogą być wywoływane z innych części programu. 
          C++
        void funkcja(int a, int b) {
          // ...
        }
        int main() {
          // ...
          funkcja(10, 20);
          return 0;
        }
Instrukcje to podstawowe jednostki kodu, które wykonują określoną akcję. 
          C++
        int main() {
          // ...
          int a = 10;
          int b = 20;
          // Dodawanie dwóch liczb
          int suma = a + b;
          // Wyświetlenie wyniku
          std::cout << "Suma wynosi: " << suma << std::endl;
        return 0;
        }

Zadanie w C++: Zamiana Wartości

Zastanów się i podaj wartości zmiennych x i y po wykonaniu poniższego kodu C++: 

#include <iostream>

int main() {
    int x = 5;
    int y = 3;
    
    x = x + y;
    y = x - y;
    x = x - y;

    std::cout << "Wartość x: " << x << std::endl;
    std::cout << "Wartość y: " << y << std::endl;

    return 0;
}
