Zmienne i Typy Danych w C++

Zmienna to miejsce w pamięci, w którym można przechowywać dane.
Typ danych określa, jaki rodzaj danych może być przechowywany w zmiennej.
W C++ istnieją dwa główne typy danych:

  • Typy proste przechowują pojedyncze wartości, takie jak liczby, tekst lub znaki.

  • Typy złożone przechowują grupy danych, takie jak tablice lub obiekty.

    • Typy proste
    Do typów prostych należą: Liczby całkowite reprezentują liczby całkowite, takie jak 1, 2, 3, itd. 
    
C++
int a = 10;
    Liczby zmiennoprzecinkowe reprezentują liczby z ułamkami, takie jak 1.23, 3.14, itd. 
    C++
float b = 1.23;
double c = 3.14;
    Znaki reprezentują pojedyncze znaki, takie jak 'a', 'b', 'c', itd. 
    C++
char d = 'a';
    Tekst reprezentuje ciąg znaków, taki jak "Ala ma kota". 
    C++
std::string e = "Ala ma kota";
    Typy złożone
    Do typów złożonych należą:
  • Tablice przechowują wiele wartości tego samego typu w jednej zmiennej.
    • C++
int tablica[5];
    Obiekty przechowują dane i funkcje w jednej strukturze. 
    C++
class Osoba {

  public:
    std::string imie;
    std::string nazwisko;

    void przedstawSie() {
      std::cout << "Mam na imię " << imie << " " << nazwisko << std::endl;
    }
};

int main() {
  Osoba osoba;
  osoba.imie = "Jan";
  osoba.nazwisko = "Kowalski";
  osoba.przedstawSie();
  return 0;
}
