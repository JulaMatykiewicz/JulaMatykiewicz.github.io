Pętle w C++: For, While, Do-While

Pętle to instrukcje, które wykonują określoną sekwencję instrukcji wielokrotnie.
W C++ istnieją trzy główne typy pętli:
Pętla for wykonuje określoną sekwencję instrukcji określoną liczbę razy.

C++
for (int i = 0; i < 10; i++) {
  // ...
}

W tym przykładzie pętla for wykona określoną sekwencję instrukcji 10 razy. 
Pierwsza instrukcja wewnątrz pętli (i = 0;) jest wykonywana tylko raz, przed rozpoczęciem iteracji pętli. 
Druga instrukcja (i < 10;) jest wykonywana przed każdą iteracją pętli, aby sprawdzić, czy pętla powinna zostać kontynuowana. 
Trzecia instrukcja (i++) jest wykonywana po każdej iteracji pętli, aby zwiększyć wartość zmiennej i o 1.
Pętla while wykonuje określoną sekwencję instrukcji, dopóki określony warunek jest spełniony.

C++
int i = 0;

while (i < 10) {
  // ...
  i++;
}

W tym przykładzie pętla while będzie wykonywać określoną sekwencję instrukcji, dopóki wartość zmiennej i będzie mniejsza niż 10.
Pętla do-while wykonuje określoną sekwencję instrukcji, dopóki określony warunek jest spełniony.
