Tablice i Ich Użycie w C++

    Tablice to struktura danych, która umożliwia przechowywanie wielu wartości tego samego typu w jednej zmiennej.
    Tablice są deklarowane za pomocą składni:

    typ nazwa_tablicy[rozmiar];

    gdzie:
    typ to typ danych przechowywanych w tablicy.
    nazwa_tablicy to nazwa tablicy.
    rozmiar to liczba wartości przechowywanych w tablicy.

    Przykład
    C++
    #include <iostream>
    int main() {
      // Deklaracja tablicy
      int tablica[5];
      // Wczytanie wartości do tablicy
      for (int i = 0; i < 5; i++) {
        std::cout << "Podaj wartość dla pozycji " << i << ": ";
        std::cin >> tablica[i];
      }
      // Wyświetlenie wartości tablicy
      for (int i = 0; i < 5; i++) {
        std::cout << "Wartość na pozycji " << i << ": " << tablica[i] << std::endl;
      }
      return 0;
    }
    Ten kod deklaruje tablicę tablica o rozmiarze 5. Następnie kod wczytuje wartości do tablicy z wejścia standardowego. 
    Na koniec kod wyświetla wartości tablicy na wyjściu standardowym.
