Wprowadzenie do Pythona: Dlaczego Wybrać Ten Język Programowania?

        Python został stworzony w roku 1990 przez Guido van Rossuma i jest to wysokopoziomowy język programowania ogólnego przeznaczenia. 
        Od 01.01.2020 r. jedyną oficjalnie rozwijaną wersją jest Python 3 i nie jest on w pełni kompatybilny z Python 2. 
        Do pracy z językiem Python potrzebny jest nam interpreter, który go obsługuje.
        W języku Python bardzo ważne są wcięcie (Tabulatory), których trzeba pilnować, aby kod działał poprawnie.

        Python ma wiele zalet:
        - prosta, przejrzysta oraz intuicyjna składnia, która sprzyja nauce,
        - dostępna, lecz niewymagana obiektowość,
        - elastyczność, związana z możliwością dodawania rozszerzeń na inne języki,
        - szeroki zestaw bibliotek,
        - otwarta licencja,
        - niezależność od platform – można go uruchomić na dowolnej platformie, która ma dostęp do interpretera Python bez potrzeby wprowadzania zmian w kodzie,
        - wysoka popularność.
