Podstawy Składni Pythona: Zmienne, Przypisanie i Operatory

        Zmienna – możemy sobie wyobrazić ją jako miejsce (np.: kontener), w którym przechowywana jest jakaś wartość. 
        Aby utworzyć zmienną potrzebujemy: nazwę zmiennej, operator przypisania oraz wartość. W języku Python możemy korzystać z następujących typów zmiennych:
        int — typ całkowity; liczby rzeczywiste
        float — typ rzeczywisty; liczby zmiennoprzecinkowe
        bool — typ logiczny; przechowuje wartość True, False
        complex — typ dla liczb zespolonych; Przechowuje liczby zespolone w formacie część_rzeczywista + część_urojona
        str — ciągi znaków

          Stwórzmy pierwszą zmienną, nazwijmy ją a oraz nadajmy jej wartość 5:
          
          a = 5

          Gdy w interpreter wpiszemy a oraz klikniemy klawisz Enter, otrzymamy wartość przypisaną do zmiennej:
          >>> a
              5
          Utwórzmy drugą zmienną b, nadajmy jej wartość 7
          b = 7
          I sprawdźmy wartości obu zmiennych, aby to zrobić, oddzielamy nazwy zmiennych przy pomocy średnika
         >>> a;b
             5
             7
          Gdy tworzymy nazwy zmiennych musimy pamiętać o:

          - rozpoczęciu literą lub znakiem podkreślenia
          - kontynuujemy cyfrą, literą lub znakiem podkreślenia
          - długość nazw jest nieograniczona
          - wielkość liter ma znaczenie
          - w języku Python występują słowa kluczowe, nazwy zmiennych nie mogą być takie same, jak one:
          False	  await	    else	    import	  Pass
          none	  break	    except	  in	      raise
          True	  class	    finally	  is	      return
          and	    continue	for	      lambda	  try
          as	    def	      from	    nonlocal	while
          assert	del	      global	  not	      with
          async	  elif	    if	      or	      yield


          Oprócz tego dobrymi praktykami przy tworzeniu nazw zmiennych są:
          - stosowanie nazw, które jasno nawiązują do zawartości np.: nazwisko, numer_domu
          - nie stosujemy polskich znaków
          - przyjęty w firmie lub zespole styl tworzenia nazw

          W języku Python oprócz poznanego już operatora przypisania „=” korzystamy z innych operatorów, które dzielimy na kilka grup:
          - arytmetyczne
          - logiczne
          - relacyjne
          - przypisania
          - identycznościowe
          - przynależności
          - bitowe
          Do wykonywania działań na liczbach korzystamy z operatorów arytmetycznych:
          + – dodawanie	 - – odejmowanie	* – mnożenie	/ – dzielenie
          >>>a = 5         >>>a = 5          >>>a = 5         >>>a = 5
          >>>b = 8         >>>b = 8          >>>b = 8         >>>b = 8
          >>>c = a + b     >>>c = a - b      >>>c = a * b     >>>c = a / b 
          >>>c             >>>c              >>>c             >>>c
          13                        -3           40                     0.625
          // – dzielenie całkowite	 % – reszta z dzielenia dwóch liczb całkowitych  ** – potęgowanie
          >>>a = 5                    >>>a = 5                          >>>a = 5
          >>>b = 8                    >>>b = 8                          >>>b = 8
          >>>c = a // b               >>>c = a % b                      >>>c = a ** b
          >>>c                        >>>c                              >>>c
          0                               5                              390625
          Operatory logiczne znajdziemy głównie w pętlach oraz instrukcjach warunkowych
          - or – lub logiczne – zwraca prawdę, gdy przynajmniej jeden z warunków jest prawdziwy, w przeciwnym razie zwraca fałsz
          - and – i logiczne – zwraca prawdę, gry wszystkie warunki są prawdziwe, w przeciwnym razie zwraca fałsz
          - not – zaprzeczenie – zaprzecza zwracaną wartość z True na False oraz z False na True
          Operatory relacyjne stosujemy, przy porównywaniu dwóch elementów, najczęściej w instrukcjach warunkowych i iteracyjnych:
          "<" - mniejszy	">" - większy	"<=" - mniejszy równy
          ">=" - większy równy	"==" - równy	"!=" - nie równy

           
          Operatory przypisania, które nadają wartości dla zmiennej znajdującej się po lewej stronie, wartości znajdującej się po stronie prawej. Oprócz operatora „=” mamy również możliwość przypisania z wykonaniem operacji matematycznej:

          a = 3	                  a += 10	   
          Zmienna a o wartości 3       	Zwiększamy wartość zmiennej a (3) o 10   
          >>>a = 3             >>>a += 10             
          >>>a                 >>>a
            3                                     13
          a -= 4	                                a **= 2	                              a //= 3
     	  Zmniejszamy wartość zmiennej a (13) o 4	Potęgujemy wartość zmiennej a (9) do 2	Dzielimy całkowicie wartość zmiennej a (81) przez 3
         >>>a -= 4                     >>>a ** 9                       >>>a //= 3
         >>>a                          >>>a                            >>>a
            9                                     81                             27       
                
          Operatory identycznościowe określają, czy dwie zmienne przechowują ten sam obiekt. Możemy również sprawdzić czy dany obiekt jest oczekiwanym typem np.: int, str
          - is – czy x jest y
          - is not – czy x nie jest y
          Operatory przynależności sprawdzają, czy dany element zawiera się w podzbiorze wartości danego obiektu.
          - in
          - not in
          W dowolnej chwili możemy zmienić wartość zmiennej: 
          >>>a = 3
          >>>a
                      3
          
          Lub podstawić wartość innej zmiennej:
          >>>a = b
          >>>a;b
                      7
                      7
          Wartością zmiennej może być również wyrażenie: 
          >>>c = 5+9*(7+3)
          >>>c
                      95
          Aby usunąć zmienną wystarczy wpisać: 
          >>>del a
          >>>a
                      Traceback (most recent call last):
                      NameError: name 'a' is not defined.
          Po sprawdzeniu, dostajemy komunikat, że zmienna a nie jest zdefiniowana.
