HTML i CSS dla początkujących: Tworzenie prostych stron internetowych

Czy kiedykolwiek zastanawiałeś się, jak są tworzone strony internetowe, które codziennie przeglądasz? Czy chciałbyś nauczyć się podstawowych umiejętności programowania, aby móc stworzyć własną stronę internetową? W takim razie HTML i CSS są idealnymi narzędziami dla Ciebie! W tym artykule przyjrzymy się tym dwóm językom programowania, które stanowią fundament większości stron internetowych, oraz pokażemy, jak zacząć tworzyć proste strony od podstaw. HTML - Język Struktury HTML, czyli HyperText Markup Language, to język używany do strukturyzowania i organizowania zawartości stron internetowych. Jego podstawowym zadaniem jest definiowanie różnych elementów na stronie, takich jak nagłówki, paragrafy, obrazy, linki itp. Poniżej znajduje się prosty przykład struktury HTML: 

            <!DOCTYPE html>
            <html lang="pl">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Moja Pierwsza Strona</title>
                <link rel="stylesheet" href="style.css">
            </head>
            <body>
                <header>
                    <h1>Witaj na mojej stronie!</h1>
                </header>
                <nav>
                    <ul>
                        <li><a href="#">Strona Główna</a></li>
                        <li><a href="#">O Nas</a></li>
                        <li><a href="#">Kontakt</a></li>
                    </ul>
                </nav>
                <section>
                    <h2>O Nas</h2>
                    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit....</p>
                </section>
                <footer>
                    <p>&copy; 2024 Moja Strona Internetowa</p>
                </footer>
            </body>
            </html>

W powyższym przykładzie zdefiniowano strukturę podstawowej strony internetowej, obejmującą nagłówek, nawigację, sekcję treści i stopkę. Elementy te są zagnieżdżone w odpowiednich tagach HTML, co nadaje strukturę i znaczenie każdemu fragmentowi. CSS - Stylowanie Strony CSS, czyli Cascading Style Sheets, to język używany do nadawania stylów i układu stronom internetowym. Pozwala on kontrolować wygląd elementów zdefiniowanych w HTML. Poniżej znajduje się prosty przykład pliku CSS, który nadaje styl wcześniej zdefiniowanym elementom: 

          body {
              font-family: 'Arial', sans-serif;
              margin: 0;
              padding: 0;
          }
  
          header {
              background-color: #333;
              color: #fff;
              text-align: center;
              padding: 1em;
          }

W powyższym przykładzie zdefiniowano style dla różnych sekcji strony, takie jak tło nagłówka, kolor czcionki, marginesy, itp. Plik CSS ten można podłączyć do strony HTML, jak pokazano w pierwszym przykładzie, za pomocą linku w sekcji . Rozwijanie Umiejętności To tylko początek podróży w świat tworzenia stron internetowych przy użyciu HTML i CSS. Istnieje wiele źródeł online, kursów i samouczków, które pomogą Ci zgłębić tajniki tych języków programowania. Pamiętaj, że praktyka to klucz do opanowania tych umiejętności, więc eksperymentuj, twórz własne projekty i nie bój się zadawać pytań społeczności programistycznej. Wkrótce zdasz sobie sprawę, że tworzenie stron internetowych to nie tylko nauka, ale również fascynująca przygoda w świecie kodowania!

Ustaw elementy w odpowiedniej kolejności, aby poprawnie załączyć arkusz stylu do strony internetowej:

  • <
  • Link
  • rel=stylesheet
  • href=style.css
  • >
