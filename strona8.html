Operatory w C++: Podstawowe Działania

        Operatory to znaki, które wykonują określone działania na danych.
        W C++ istnieją dwa główne typy operatorów:
        
  • Operatory arytmetyczne
    • 
        
  • Operatory logiczne
    • 
        Operatory arytmetyczne wykonują następujące operacje:
        
  • Dodawanie: +
    • 
        
        C++
        int a = 10;
        int b = 20;
      
        int suma = a + b; // suma = 30
      
    
  •  Odejmowanie: -
    • 
        
        C++
        int a = 10;
        int b = 20;
        
        int roznica = a - b; // roznica = -10
      
        
    
  •  Mnożenie: *
    • 
        
        C++
        int a = 10;
        int b = 20;
        
        int iloczyn = a * b; // iloczyn = 200
      
    
  •  Dzielenie: /
    • 
        
        C++
        int a = 10;
        int b = 20;
        
        float iloraz = a / b; // iloraz = 0.5
      
    
  •  Reszta z dzielenia: %
    • 
        
        C++
        int a = 10;
        int b = 20;
        
        int reszta = a % b; // reszta = 10
      
        
    Operatory logiczne
    
        Operatory logiczne wykonują następujące operacje:
        
  • Równość: ==
    • 
        
        C++
        int a = 10;
        int b = 20;
        bool rownosc = a == b; // rownosc = false
      
        
  • Nierówność: !=
    • 
        
        C++
        int a = 10;
        int b = 20;
        bool nierownosc = a != b; // nierownosc = true
      
        
  • Większe niż: >
    • 
        C++
        int a = 10;
        int b = 20;
        
        bool wieksze = a > b; // wieksze = false
        
        
  • Mniejsze niż: <
    • 
        C++
        int a = 10;
        int b = 20;
        
        bool mniejsze = a < b; // mniejsze = true
        
      
  • Większe lub równe niż: >=
    • 
        C++
        int a = 10;
        int b = 20;
        
        bool wieksze_rowne = a >= b; // wieksze_rowne = false
        
      
  •  Mniejsze lub równe niż: <=
    • 
        C++
        int a = 10;
        int b = 20;
        
        bool mniejsze_rowne = a <= b; // mniejsze_rowne = true
