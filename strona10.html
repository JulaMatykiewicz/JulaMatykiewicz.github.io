Warunki i Instrukcje Warunkowe w C++

        Instrukcje warunkowe to instrukcje, które wykonują określoną akcję w zależności od wartości warunku.
        W C++ istnieją dwa główne typy instrukcji warunkowych:

        Instrukcja if-else wykonuje określoną akcję, jeśli określony warunek jest spełniony, a inną akcję, jeśli warunek nie jest spełniony.

        C++
        int a = 10;
        if (a > 10) {
          // ...
        } else {
          // ...
        }

        W tym przykładzie instrukcja if-else wykona określoną akcję, jeśli wartość zmiennej a będzie większa niż 10. 
        Jeśli warunek nie jest spełniony, instrukcja else zostanie wykonana.
        Instrukcja switch wykonuje określoną akcję w zależności od wartości zmiennej.

        C++
        int a = 1;
        switch (a) {
          case 1:
            // ...
            break;
          case 2:
            // ...
            break;
          default:
            // ...
            break;
        }

        W tym przykładzie instrukcja switch wykona określoną akcję w zależności od wartości zmiennej a. 
        Jeśli wartość zmiennej a będzie równa 1, zostanie wykonana instrukcja case 1. 
        Jeśli wartość zmiennej a będzie równa 2, zostanie wykonana instrukcja case 2. 
        W przeciwnym razie zostanie wykonana instrukcja default.
